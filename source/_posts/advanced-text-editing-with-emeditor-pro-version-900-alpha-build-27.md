EmEditor Professional 9.00 alpha 27 is available for download. Updates from  
 v9.00 alpha 26 include:  
 – New tab in the Customize dialog: “View” allows you to change the Customize bar colors and whether to change font size with CTRL + Mouse Wheel.  
 – UTF-16 entry in the Auto Detect encoding dialog box.  
 Please download  
[http://www.emeditor.com/pub/emed900a27epx.msi](https://tools.techidaily.com/emeditor/products/) (English, 32-bit)  
[http://www.emeditor.com/pub/emed900a27ep64.msi](https://tools.techidaily.com/emeditor/products/) (English, 64-bit)  
 The portable version can be produced from the Import/Export wizard on the Tools menu after you use the installer. However, the portable version runs slower than the one using the Registry.  
 The Help has not been updated. Please let me know earlier rather than later if you have any bug reports or comments. Thanks!
